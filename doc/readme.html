jStyleParser project

What is jStyleParser?

jStyleParser parses CSS 2.1 into structures which can be assigned to DOM elements. While handling errors, it is user agent conforming according to specification.

Compilation prerequisites

  • ANTLR 3.1 to generate Lexer and Parser from grammar files. Generated output is part of the project to make compilation easier.
  • ANT to compile classes into jar package by using provided build file
  • Libraries
    • antlr-runtime for parsing CSS
    • any DOM and DOM Traversal implementation. Currently, jStyleParser has been tested with NekoHTML parser and Xerces DOM implementation
    • slf4j and arbitrary implementation for logging subsystem
    • (Optionally) JUnit with Hamcrest core extension (4.4) for JUnit tests
    All required libraries are provided within SVN repository.
